# Everyx — Кафетерий льгот (демо)

Интерактивное демо платформы гибких льгот: продающий лендинг, личный кабинет
сотрудника и панель администратора C&B.

- Лендинг — `index.html`
- Приложение — `app.html` (роли «Сотрудник» / «HR · C&B»)

Статический сайт (HTML/CSS/JS), без сборки. Дизайн-система Burnt Coral.
Демонстрационная версия — компания и сотрудники вымышлены.
